Kontent.ai Helper Tools

Unofficial tools for Kontent.ai

User & Asset Data Management

Subscription User Export - Exports user data using the Subscription API.
Export Asset Info - Extracts basic asset information using the Management API.
Asset Calculator - Counts assets in a subscription or environment.
Items Calculator - Counts items in the environment.
Word Count - Counts words within the environment's items.

Item & Asset Discovery

Find Items - Finds items based on their element types.
Search - Searches for items based on a string in their elements.
Kontent Lens - Compares all image assets against an uploaded image.

API & UI Linking

Link to the UI from the Codename - Generates a UI link from a given codename.
UI Link to Delivery API - Retrieves a Delivery API response from a UI content item link.

Custom Elements & Webhooks

YouTube Search - A custom element enabling YouTube search.
Schedule Unpublish - A webhook script for scheduling the unpublishing of a content item.

Others

VBA Script for Filtering Email Domains - A VBA script for filtering email domains in Excel.
VBA Script for Filtering Environments - A VBA script for filtering Environment IDs in Excel report.
